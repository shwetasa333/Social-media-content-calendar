## 📌 Project Title:
30-Day Social Media Content Calendar for TrendStyle

## 📖 Project Description:

This project includes a 30-day social media content calendar 
designed for Instagram, LinkedIn, and Twitter/X platforms.

The calendar includes:

- Post types (Reel, Carousel, Story, Text)
- Captions
- Hashtags
- Posting schedule
- Sample Canva creatives

## 🛠 Tools Used:

- Canva (for designing creatives)
- Microsoft Excel (for content calendar)
- GitHub (for project hosting)

## 📂 Files Included:

- Social Media Content Calendar.xlsx  
- Hashtag_Research.docx  
- Canva Creatives (PNG files)  

## 🎯 Objectives:

- Plan structured social media content
- Improve brand engagement strategy
- Practice creative design skills
- Learn GitHub project management
